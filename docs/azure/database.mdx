Each environment gets a dedicated Azure Database instance sized appropriately for the workload, with automatic backups, encryption at rest, and secure network configuration.

## Instance Configuration

### Instance Sizing by Environment

<Tabs>
  <Tab title="Development">
**SKU:** `B_Gen5_1`

**Specifications:**
- vCores: 1
- RAM: 2 GB
- Storage: 32 GB
- IOPS: 640

**Configuration:**
```hcl
resource "azurerm_postgresql_flexible_server" "main" {
  name                = "myapp-dev-db"
  resource_group_name = azurerm_resource_group.main.name
  location            = azurerm_resource_group.main.location
  
  sku_name   = "B_Standard_B1ms"
  storage_mb = 32768
  version    = "15"
  
  administrator_login    = "psqladmin"
  administrator_password = random_password.db_password.result
  
  zone                      = "1"
  backup_retention_days     = 7
  geo_redundant_backup_enabled = false
  
  high_availability {
    mode = "Disabled"
  }
}
```

**Monthly Cost:** ~$30

## Database Security

### Encryption at Rest

```hcl
resource "azurerm_postgresql_flexible_server" "main" {
  # ... other configuration ...
  
  customer_managed_key {
    key_vault_key_id = azurerm_key_vault_key.db.id
  }
}
```

### Password Management

```hcl
resource "random_password" "db_password" {
  length  = 32
  special = true
}

resource "azurerm_key_vault_secret" "db_password" {
  name         = "myapp-dev-db-password"
  value        = random_password.db_password.result
  key_vault_id = azurerm_key_vault.main.id
}
```
